# Hướng Dẫn Sử Dụng

## Dịch Thuật

1. Nhập văn bản cần dịch
2. Chọn ngôn ngữ nguồn và đích
3. Chọn giọng văn (tự nhiên, thân mật, trang trọng)
4. Nhấn "Dịch"

## Từ Điển

- Click vào từ trong kết quả dịch để xem nghĩa
- Hỗ trợ phát âm với Web Speech API

## Lịch Sử

- Tất cả bản dịch được lưu trong localStorage
- Tìm kiếm lịch sử với thanh tìm kiếm

## Phím Tắt

- `Ctrl+Enter` - Dịch ngay
- `Ctrl+Shift+D` - Mở từ điển
