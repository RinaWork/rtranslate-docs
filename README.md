## GitHub Pages Setup

1. Go to repo Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` / `root`
4. Theme: Choose a Jekyll theme (optional)
5. Save

## Contributing

1. Create/edit `.md` files in appropriate folder
2. Commit and push - GitHub Pages auto-updates
