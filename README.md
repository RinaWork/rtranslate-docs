# Documentation

Documentation source files and viewer.

## Structure

- `docs-viewer/` - React app to view these docs
- `architecture/` - System architecture docs
- `api/` - API reference
- `deployment/` - Deployment guides
- `user/` - User guides (VI/EN)

## Quick Start

```bash
cd docs-viewer
npm install
npm run dev
```

## Adding Documentation

1. Create `.md` files in appropriate folder
2. The docs viewer auto-discovers new files
3. Supports GitHub-flavored markdown
